Sobre el proyecto

ArchivoOCR es una plataforma desarrollada para la digitalización, transcripción y gestión de documentos históricos. Combina tecnología de reconocimiento óptico de caracteres con revisión humana para ofrecer transcripciones de alta calidad.

El sistema preserva tanto el facsimilar original como el texto transcrito, permitiendo la búsqueda a texto completo y la descarga en formatos estándar.
